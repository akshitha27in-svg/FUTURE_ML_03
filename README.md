# Resume Screening System - Jupyter Notebook Project

This submission is notebook-first. Open the notebook below in Jupyter Notebook, JupyterLab, or VS Code:

```text
notebooks/resume_screening_jupyter_project.ipynb
```

The notebook includes:

- Resume text preprocessing
- Required skill extraction
- Job description parsing
- TF-IDF resume-to-job similarity scoring
- Candidate ranking
- Missing skill identification
- Visual score comparison
- Visual skill-match heatmap
- Recruiter-friendly explanations

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```
